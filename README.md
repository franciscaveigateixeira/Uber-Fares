# Uber fares — Streamlit explorer

Simple Streamlit app to explore Uber fares and cluster fare vs distance.

How to run locally

1. (Optional) create a virtual environment and activate it.
2. Install dependencies:

```bash
python -m pip install -r requirements.txt
```

3. Launch the app:

```bash
streamlit run uber_cluster_app.py
```

Then open http://localhost:8501 in your browser.

If the app asks for email on first run in an interactive terminal, just press Enter to skip.
